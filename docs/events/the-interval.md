# The Interval

<span class="entity-badge entity-badge--dark">Event — Recurring Threat</span>

The span of time between attempted **[Archfiend](../beings/the-archfiend.md)** risings, measured from the appearance of the **black cloud** to the next. The observed **Interval** is approximately **forty-seven years**.

For full details, see **[The Interval — History](../lore/the-interval.md)**.
